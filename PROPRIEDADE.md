# PartyMix — Aviso de propriedade

## Resumo

O PartyMix **não é software livre / open source**. É um produto comercial em desenvolvimento.

## O que não é permitido (sem autorização escrita)

- Copiar ou clonar o repositório para criar outra app na Play Store ou App Store
- Publicar o código em GitHub público
- Vender ou distribuir o código-fonte
- Reutilizar cartas, textos, nome **PartyMix** ou UI como produto concorrente

## O que podes fazer (como autor / equipa autorizada)

- Desenvolver e testar localmente
- Publicar **tu** na Play Store como dono do projeto
- Monetizar com anúncios, IAP ou subscrição
- Criar packs de conteúdo premium (`data/*.json`)

## Proteção prática antes da loja

1. Repositório **privado** (GitHub/GitLab)
2. Não enviar `partymix_for_claude.zip` nem código a terceiros
3. `ADMIN_PASSWORD`, `JWT_SECRET` e `GROQ_API_KEY` só no servidor
4. Registar marca/nome quando fores publicar (opcional mas recomendado em Portugal/UE)

## Publicação na Play Store

O utilizador final instala um **APK/AAB**, não o teu repo. Quem “clona” o teu Git não tem o teu backend, a tua base de dados nem os teus packs — desde que não publiques o código abertamente.
