# 🎉 PartyMix

App de jogos de festa para reunir amigos, casais e família.

## 🎮 Modos de Jogo
- 💕 Casal
- 🎉 Amigos
- 🏡 Família
- 🃏 Cartas
- 👁️ Mister White

## 🚀 Stack
- React + Vite
- Node.js + Express
- MongoDB
- Netlify + Render
